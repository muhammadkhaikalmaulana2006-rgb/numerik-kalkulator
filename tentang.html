TENTANG APLIKASI

Aplikasi ini dibuat untuk membantu proses perhitungan numerik berbasis web.

📖 User Manual

1. Login menggunakan akun yang tersedia.

2. Pilih metode pada menu utama.

3. Masukkan data yang diketahui.

4. Klik tombol HITUNG.

5. Hasil, tabel iterasi, dan grafik akan muncul.

6. Gunakan tombol RESET untuk menghitung ulang.
